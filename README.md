# nsml-submit-bot
Auto-submit bot for Naver AI-Vision Hackathon 2018

혹시나 밤새 1시간 마다 제출하실 분이 있으실 것 같아,  
자동 제출을 할 수 있는 코드를 만들어 보았습니다.

## How to run.

```
python main.py
```

그리고 메인 코드에 맨 상단에 있는 변수를 수정하셔서  
원하시는 세션/모델 을 제출하시면 됩니다.


## Requirements.

* 미리 콘솔로 nsml 로그인이 되어 세션을 열어두셔야 합니다.
* python 3.6+
